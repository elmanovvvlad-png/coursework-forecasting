# coursework-forecasting
# Курсовая работа: Прогнозирование с помощью тренд-сезонных моделей

## О проекте
Данный репозиторий содержит код и материалы курсовой работы по дисциплине «Автоматизированные системы прогнозирования». Работа выполнена студентами группы ББИ2304 МТУСИ.

**Тема:** Прогнозирование с помощью тренд-сезонных моделей

**Цель работы:** Разработка и применение тренд-сезонной модели для анализа и прогнозирования временных рядов на примере данных о производстве мяса крупного рогатого скота.

## Используемые методы
- Модель Хольта-Уинтерса (метод Холта — частный случай без сезонной компоненты)
- Аддитивная модель тренда
- Визуализация временных рядов
- Оценка качества прогноза (MAPE)

## Данные
Временной ряд — годовые данные о производстве мяса крупного рогатого скота (говядина и телятина парная, остывшая или охлаждённая) за период 2017–2021 гг. Источник: Росстат.
